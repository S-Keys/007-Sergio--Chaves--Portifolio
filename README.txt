# APRESENTAÇÃO PARA LIBRARIA
  
Página de apresentenção de habilidades em LIBRAS, edição de vídeo e frot end.
 
## Índice
 
- [Visão Geral](#visão-geral)
- [Pré-Requisitos](#pré-requisitos)
- [Contribuição](#contribuição)
 
## Visão Geral
 
O grupo Libraria, como parte de seu processo seletivo, solicita a gravação de vídeos de interpretação de uma notícia em português para LIBRAS, para pessoas interessadas em atuar como intérpretes para LIBRAS, gravar um áudio em português para fazer dublagem de surdos e um vídeo teste editado, para aqueles que têm interesse em atuar como editores de vídeo. Todas estas funções me interessam, mas pensei em ir além e fazer esta página para apresentar minhas habilidades como desenvolvedor front end e incluir nela minha interpretação, dublagem e vídeos editados, fazendo assim um combo de todas as minhas habilidades.
